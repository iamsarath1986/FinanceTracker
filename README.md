# FinanceTracker

FinanceTracker is an early-stage financial application project planned with a Vue.js frontend and a FastAPI backend.

The goal is to build a personal finance app for tracking accounts, transactions, budgets, categories, and financial summaries.

## Planned Tech Stack

- Frontend: Vue 3, Vite, TypeScript
- Backend: FastAPI, Python
- Database: PostgreSQL for production, SQLite optional for local development
- ORM/Migrations: SQLAlchemy or SQLModel with Alembic
- Authentication: JWT-based login
- Testing: Pytest for backend tests

## Planned Features

- User registration and login
- Account management
- Income and expense categories
- Transaction tracking
- Budget management
- Dashboard summaries
- Spending reports and charts
- CSV import and export

## Current Status

This project is currently in the planning and setup stage. The existing `main.py` file is a starter Python script generated by PyCharm.

## Local Development

The final local development setup will likely include separate backend and frontend commands:

```bash
# Backend
cd backend
uvicorn app.main:app --reload

# Frontend
cd frontend
npm install
npm run dev
```

These commands will be updated once the backend and frontend applications are scaffolded.
